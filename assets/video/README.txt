VEROX 视频放置说明
====================

把 mp4 视频放进本文件夹（assets/video/），然后在后台“视频”里或
content/videos.json 中把 src 填成相对路径，例如：

  assets/video/apex-drive.mp4
  assets/video/results.mp4
  assets/video/lab.mp4

说明：
- 只放封面（poster）也能正常上线，点击后才需要 mp4 才能播放。
- 短、自然、竖屏或 16:10 的视频转化最好（使用演示 / 真实评价）。
- 文件别太大，建议单条 < 15MB，必要时压缩。

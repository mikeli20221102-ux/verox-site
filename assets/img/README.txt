VEROX 图片放置说明
====================

把真实图片放进本文件夹（assets/img/），文件名按下面约定即可自动覆盖占位图，
无需改代码。放入前网站会自动显示占位图（picsum 随机图），不会破图。

首页 / 通用：
  hero.jpg           首页大图（建议竖图，约 720×820）
  about-brand.jpg    “关于”板块配图（约 680×760）
  shipping.jpg       “配送/保障”板块配图（约 640×720）

产品（与 content/products.json 的 key 对应）：
  <key>-1.jpg ... <key>-4.jpg   产品封面与相册图（约 600×450 或更高）
  例如：
  apex-drive-1.jpg / apex-drive-2.jpg / apex-drive-3.jpg / apex-drive-4.jpg
  iron-surge-1.jpg ...
  tongkat-pure-1.jpg ...
  maca-energy-1.jpg ...
  night-recharge-1.jpg ...
  stay-power-1.jpg ...

视频封面（与 content/videos.json 对应）：
  apex-drive-cover.jpg / results-cover.jpg / lab-cover.jpg

建议：产品图用白底或统一背景，风格一致；真实产品图是转化的关键，务必替换。
